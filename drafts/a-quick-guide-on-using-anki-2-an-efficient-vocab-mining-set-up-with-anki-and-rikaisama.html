So you want to:

  • read a Japanese novel and don’t end up crying yourself to sleep realizing you know nothing, Jon Snow?
  • learn Japanese by watching anime, but like, for real?
  • create beautiful Anki sets filled with personally selected vocabulary AND crystal-clear audio without spending days of mindnumbing copypasting?

Welcome to the lovely world of Integrated Anki & Rikaisama!

We’ve already established Rikaichan/kun as an incredibly helpful tool for reading Japanese texts online, yet one thing seemed missing still: a seemless integration with Anki for efficient vocabulary mining. A recent extension of the original Rikaichan, aptly named Rikaisama (it’s a pun!), adds several new features including communication with the Anki API and J-Pod101’s1 audiofiles. Concretely, this means audio playback of tens of thousands of words and expressions as well as easy integration of new vocab into Anki. This blog serves as a brief tutorial on setting up this environment as well as demonstrating some useful appliances.
